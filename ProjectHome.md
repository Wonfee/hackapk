# HACKAPK  V1.1 #


# 功能简介 #

> 本软件套装是为了快速的对ANDROID软件进行逆向工程，汉化，以及分析软件的设计和实现而收集整理。

> 目前本软件集成了 JD-GUI,DEX2JAR,APKTOOL

# 下载地址 #

http://download.csdn.net/detail/vrix/7129161

# 用法 #

windows: version 1.0

> winhackapk.bat target.apk outputdir

linux:  version 1.1

> . hackapk.sh  target.apk
